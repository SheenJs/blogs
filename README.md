## 目录解析

```text
└─docs
    ├─.vitepress  vitepress相关配置文件夹
    │  ├─cache  打包缓存
    │  │  └─deps
    │  ├─sidebar  侧边栏配置
    │  └─theme  主题配置
    │      └─components  全局组件
    │          └─src
    ├─vitepress 博客
    └─public  公共静态资源文件
```

## 项目运行

```bash
git clone https://gitee.com/guanghuijs/blogs
cd blogs
pnpm i
pnpm dev
````
